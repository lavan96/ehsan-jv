eCOLE Learning Exchange 11.0

Empowering Modern Education

Key Features

eCOLE consists of a comprehensive set of tools that enable the institute to cover a range of activities from Student Enrolments to Payment Management to Attendance Rosters. eCOLE automates the activities of an Education Institution such as Student Registrations, Employee Management, Student's Fee generation, Prospect Management, Medical Check-up details, Attendance entry, Marks entry, Library Management, Inventory of Books, Transportation. The system supports multiple-currency and multiple-location, and can be used by on a multiple-user setup mode with Group Level and User level rights.
